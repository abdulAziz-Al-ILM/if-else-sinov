Mavzu: Shartli Operatorlar (if-else)

Borliqni mantiq bilan boshqarish

"Iroda — bu 'if' (shart) qo'yish qobiliyatidir. Agar sen o'z hayotingga shart qo'ymasang, dunyo senga 'else' (majburiy variant)ni beradi."
Mizan izohi: Hayotda va dasturlashda hech narsa o'z-o'zidan hal bo'lmaydi. Biz kod orqali kompyuterga qoida (if) o'rnatamiz. Bu bizning irodamiz. Agar biz qoida yozmasak yoki xato yozsak, tizim avtomatik ravishda boshqa tomonga, ya'ni biz xohlamagan aks holda (else) holatiga tushib qoladi. Dasturlash — bu nazoratni qo'lga olishdir.

Akademik Ma'ruza (Nazariy Asoslar)

Algoritmik Shoxlanish (Branching): Dastur kodi doim tepadan pastga qarab chiziqli ishlaydi. if-else konstruksiyasi esa bu chiziqni ikkiga bo'ladi. Protsessor bir vaqtning o'zida ikkala yo'ldan yurolmaydi, u faqat bittasini tanlashi shart.
Case-Sensitivity (Registrga sezgirlik): C# tili o'ta qat'iydir. Katta "A" va kichik "a" xotirada turli xil baytlarda saqlanadi. Shuning uchun "Yur" va "yur" C# uchun mutlaqo begona so'zlardir.

Kod Anatomiyasi: Vazifalar taqsimoti

Kodni shunchaki yozuvlar emas, balki birga ishlayotgan kichik robotlar tizimi deb tasavvur qiling:

using System; "Ruxsatnoma": Zavod ishlashi uchun koinot qonunlarini (bazaviy kutubxonalarni) ulash. Busiz hech bir robot ishlamaydi.
string buyruq = "Quti": Foydalanuvchi yozgan so'zni saqlab turadigan vaqtincha xotira idishi.
Console.ReadLine() "Quloq Roboti": Uning yagona vazifasi — jim turib tashqaridan (klaviaturadan) kelayotgan ma'lumotni eshitish.
if (buyruq == "Yur") "Qorovul Roboti": Qutidagi so'zni tekshiradi. Faqat aniq mos kelsagina darvozani ochadi.
Console.WriteLine() "Jarchi Roboti": Natijani ekranga ovoza qiluvchi mexanizm. 
using System;

namespace IlmMizan
{
    class RobotFactory
    {

        {
            Console.Write("Buyruq kiriting (Yur/To'xta): ");

Eslatma: C# Case-Sensitive. Aynan "Yur" deb katta harf bilan yozib ko'ring.

> Dastur ishga tushdi. Buyruq kutilmoqda...
Odam Tili: "Men senga savol beraman. Eshitgan javobimni 'buyruq' qutisiga solaman. Agar u qutining ichida aynan kattasi bilan 'Yur' degan so'z bo'lsa qadam tashlayman. Boshqa har qanday holatda (masalan, 'yur', 'qoch', 'salom') joyimda kutaman."
Texnik Tarjima:
ReadLine() - Foydalanuvchi kiritgan ma'lumotni kiritish oqimidan o'qiydi.
if (a == b) - Ikkita string qiymatni qattiq (strict) solishtiradi. Registr (Case) mos kelishi shart.
else - Boolean natija False bo'lgandagi default (standart) yechim.

Mustaqil Amaliyot (Topshiriqlar)

1. "Katta-Kichik xato"

Nega "yur" deb kichkina harfda yozsak tizim uni qabul qilmayapti? Bu xatoni oldini olish uchun C# da qanday metod borligini qidirib toping (Maslahat: .ToLower() metodini o'rganing).

2. Uchlik tanlov

Kodga else if qo'shing. Agar buyruq "Orqaga" bo'lsa, robot orqaga qaytishi kerakligi haqida mantiq yozing.

3. Matematik Shart

String o'rniga Integer (int) ishlating. Foydalanuvchidan yoshini so'rang. Agar yoshi 18 dan katta bo'lsa tizimga kiritsin.
