4-Mavzu: if-else — Qaror qabul qilish mantiqi

Robotimizni o'ylashga majbur qilamiz

Scenario: "Svetofor Roboti"

Tasavvur qiling, robotimiz yo'l chetida turibdi. Agar chiroq qizil bo'lsa — u to'xtashi kerak, aks holda (agar qizil bo'lmasa) — u harakatlanishi mumkin. Bu mantiq C# da qanday yoziladi?

C# SOURCE CODE
string chiroq = "Qizil";

if (chiroq == "Qizil") 
{
    Console.WriteLine("To'xtang!");
}
else 
{
    Console.WriteLine("Yuring!");
}
TARJIMA VA MANTIQ

Insoniy (Sodda) Talqin:

1. Kelin, chiroqning rangini "Qizil" deb belgilaylik.

2. Agar chiroq chindan ham qizil bo'lsa...

3. ...Ekranga "To'xtang!" degan buyruqni chiqaramiz.

4. Aks holda (qizil bo'lmasa)...

5. ..."Yuring!" deb ruxsat beramiz.

Texnik (Literal) Talqin:

string: Xotiradan matn uchun joy ajratish.

if: Shartli operator boshlanishi (Boolean true/false tekshiruvi).

==: Taqqoslash operatori (Tenglikni tekshirish).

else: Shart false (noto'g'ri) chiqqanda ishlaydigan blok.

{ }: Kod blokining chegaralari (Scope).

Har bir elementga ta'rif:

If operatori: Bu dasturning "miyasidagi" birinchi shubha. U o'zidan keyingi qavs ichidagi narsani tekshiradi. Agar natija "True" bo'lsa, kod ichiga kiradi.
Boolean (bool): Shartli operatorlar faqat ikkita tilni tushunadi: Rost (True) yoki Yolg'on (False). O'rtacha variant yo'q.
Else (Aks holda): Bu — "Plan B". Agar `if` sharti bajarilmasa, dastur avtomatik ravishda bu blokga tushadi. `else` o'zidan keyin shart talab qilmaydi.
Taqqoslash belgilari:
`==` (Tengmi?), `!=` (Teng emasmi?), `>` (Kattami?), `<` (Kichikmi?).
