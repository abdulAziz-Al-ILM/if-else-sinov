Tizim tayyor...
if Operator

Dasturning asosiy "tanlov" nuqtasi. U faqat rost (true) natijani qidiradi.

== (Comparison)

Qiymat berish emas, solishtirish. "Ikkala tomon bir-biriga tengmi?" deb so'raydi.

else (Alternative)

Agar birinchi shart bajarilmasa, dastur yiqilmasligi uchun "B reja" vazifasini o'taydi.

Boolean Logic

Svetofor faqat ikki holatda bo'lishi mumkin: Shart bajarildi yoki bajarilmadi.
