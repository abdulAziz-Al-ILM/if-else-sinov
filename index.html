4-Mavzu: Shartli Operatorlar (if-else)

Irodaning mantiqiy algoritmi

"Iroda — bu if (shart) qo'yish qobiliyatidir. Agar sen o'z hayotingga shart qo'ymasang, dunyo senga else (majburiy variant)ni beradi."

1. Robotni "Eshitishga" o'rgatamiz

// Robot bizdan buyruq kutmoqda
Console.Write("Buyruqni kiriting: ");
string buyruq = Console.ReadLine();

if (buyruq == "Yur") 
{
    Console.WriteLine("Robot harakatga keldi!");
}
else 
{
    Console.WriteLine("Robot hali ham kutmoqda...");
}
