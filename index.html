C# if-else Operatorlari

Kod orqali tanlov qilish san'ati

Scenario: "Robot va Svetofor"

Mantiq oddiy: Agar chiroq "Qizil" bo'lsa, robot to'xtab turishi shart. Aks holda (agar "Yashil" yoki boshqa rang bo'lsa), robot harakatini davom ettirishi mumkin. Biz kod orqali bu mantiqni robotning "miyasiga" joylaymiz.

string chiroqRang = "Qizil";

if (chiroqRang == "Qizil") 
{
    Console.WriteLine("Robot: To'xtab turibman.");
}
else 
{
    Console.WriteLine("Robot: Harakatlanyapman.");
}

Tarjima va Mantiq

1. Kelin, chiroqni "Qizil" deb olaylik.

2. Agar chiroq chindan ham Qizil bo'lsa...

3. ...robot To'xtab turadi.

4. Aks holda (Qizil bo'lmasa)...

5. ...robot Yurib ketadi.

if: Shartli operator, Boolean natijani (true/false) tekshiradi.

==: Taqqoslash operatori, tenglikni solishtiradi.

else: Shart `false` chiqqanda bajariladigan blok.

{ }: Kod Scope (chegarasi).

3D Mantiqiy Sinov: 'if-else' amalda

Har bir funksiyaga ta'rif:

if(shart) { }: Dasturning eng katta shubhasi. U qavs ichidagi shart rost (true) bo'lsagina keyingi jingalak qavs ichidagi kodni bajaradi.
== (Taqqoslash): "Tengmi?" deb so'raydi. Bitta `=` (qiymat berish) bilan adashtirmaslik kerak. `x = 5` (x ga 5 qiymatini ber), `if(x == 5)` (x 5 ga tengmi?).
else { }: Bu "Plan B". Agar `if` sharti bajarilmasa, dastur avtomatik ravishda bu blokga tushadi. `else` o'zidan keyin yangi shart talab qilmaydi.
