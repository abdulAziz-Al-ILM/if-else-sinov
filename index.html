Mavzu: Shartli Operatorlar (if-else)

Borliqni mantiq bilan boshqarish

"Iroda — bu 'if' (shart) qo'yish qobiliyatidir. Agar sen o'z hayotingga shart qo'ymasang, dunyo senga 'else' (majburiy variant)ni beradi."
Mizan izohi: Hayotda va dasturlashda hech narsa o'z-o'zidan hal bo'lmaydi. Biz kod orqali kompyuterga qoida (if) o'rnatamiz. Bu bizning irodamiz. Agar biz qoida yozmasak, tizim avtomatik ravishda boshqa tomonga, ya'ni biz xohlamagan aks holda (else) holatiga tushib qoladi. Dasturlash — bu jarayonlar ustidan nazoratni qo'lga olishdir.

Akademik Ma'ruza (Nazariy Asoslar)

Algoritmik Shoxlanish (Branching): Dastur kodi doim tepadan pastga qarab chiziqli ishlaydi. if-else bu chiziqni ikkiga bo'ladi. Protsessor bir vaqtning o'zida ikkala yo'ldan yurolmaydi, u faqat bittasini tanlashi shart.
Case-Sensitivity (Registrga sezgirlik): C# tili o'ta qat'iydir. Katta "A" va kichik "a" xotirada turli baytlarda saqlanadi. "Yur" va "yur" C# uchun mutlaqo xilma-xil buyruqlardir. Dasturchi bunga juda ehtiyot bo'lishi lozim.

Kod Anatomiyasi: Robotlar Ekipaji

Kodni shunchaki yozuvlar emas, balki birga ishlayotgan kichik robotlar tizimi deb tasavvur qiling:

using System; "Ruxsatnoma": Koinot qonunlarini (bazaviy kutubxonalarni) ulash. Busiz hech bir robot ishlamaydi.
string buyruq = "Quti": Foydalanuvchi yozgan so'zni saqlab turadigan vaqtincha xotira idishi.
Console.ReadLine() "Quloq Roboti": Uning yagona vazifasi — jim turib tashqaridan (klaviaturadan) ma'lumot kelishini kutish.
if (buyruq == "Yur") "Qorovul Roboti": Qutidagi so'zni tekshiradi. Faqat aniq mos kelsagina o'ziga tegishli qavslar { } ichiga ruxsat beradi.
Console.WriteLine() "Jarchi Roboti": Qabul qilingan qaror natijasini ekranga ovoza qiluvchi mexanizm.

Eslatma: Qat'iy "Yur" deb katta harf bilan yozib ko'ring.

> Dastur ishga tushdi. Buyruq kutilmoqda...
Odam Tili: "Men senga savol beraman. Eshitgan javobimni 'buyruq' qutisiga solaman. Agar u qutining ichida aynan kattasi bilan 'Yur' degan so'z bo'lsa qadam tashlayman. Boshqa har qanday holatda kutaman."
Texnik Tarjima:
ReadLine() - Foydalanuvchi kiritgan ma'lumotni kiritish oqimidan o'qiydi.
if (a == b) - Ikkita string qiymatni qattiq (strict) solishtiradi.
else - Boolean natija False bo'lgandagi holat.

Mustaqil Amaliyot (Topshiriqlar)

1. "Katta-Kichik xato"

Nega "yur" deb kichkina harfda yozsak tizim uni qabul qilmayapti? Bu xatoni oldini olish uchun C# da qanday metod borligini qidirib toping (Maslahat: .ToLower()).

2. Uchlik tanlov

Kodga else if qo'shing. Agar buyruq "Orqaga" bo'lsa, robot orqaga qaytishi kerakligi haqida mantiq yozing.

3. Matematik Shart

Foydalanuvchidan yoshini so'rang. Agar yoshi 18 dan katta bo'lsa tizimga kiritsin.
